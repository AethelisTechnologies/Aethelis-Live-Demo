EHR MESH TRANSPORT LAYER — OPTION 2
Resilient EHR Data Relay via Aethelis Mesh
Aethelis acts as the transport layer — not the data layer. During a network outage, the hospital's existing EHR system routes through the mesh. Aethelis moves packets. It never stores, reads, or modifies patient data.
EHR RELAY SESSIONS — SIMULATED
HOW IT WORKS
1
Hospital EHR detects network outage and routes traffic to Aethelis gateway
2
Aethelis mesh nodes relay encrypted HL7/FHIR packets between departments
3
Data arrives at destination EHR system intact — Aethelis never opens the payload
✓
No patient data stored on Aethelis nodes. No HIPAA scope expansion.
COMPLIANCE IMPACT
HIPAA Info Blocking risk ELIMINATED
EHR availability during outage MAINTAINED
Patient data on Aethelis nodes NONE
